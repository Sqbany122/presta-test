Product Status module

Statuses for products in orders

All products in orders, can have personal statuses. This helps trace fulfillment of an order for each product individually.

1. New tab 'Product Status' with fields:

    ID product
    ID order
    Customer
    Product name
    Product price
    Product status
    Date

2. Filter
3. Can change statuses directly from filst, withuot reloading page (ajax)
4. Use standart status list from Prestashop.
5. New block in order page with statuses of products in current order.
